# Typing Test

A terminal-based typing speed test written in Python. For the general functionality of the test, I followed Tech With Tim's tutorial. (https://youtu.be/NpmFbWO6HPU?si=2FlI3uCk8oImiPAT)
Below, I've outlined the features that I added on my own.

## Features
- Multiple difficulty levels to test (easy, intermediate, hard)
- Tracks accuracy while the user types.

## Installation
1. Install Python 3
2. Clone the repository
3. Run the program: typing_test.py


## Files
- typing_test.py → main program
- easy.txt → easy typing text
- intermediate.txt → medium difficulty text
- hard.txt → hard difficulty text

## Future Improvements

- Add timer
- Storage of WPM and accuracy to compare past scores
